ℹ️ About This Application

🤖 How It Works

  1. Text Preprocessing: Cleans and normalizes input text
  2. Stop Word Removal: Removes common filler words
  3. Stemming: Reduces words to their root forms
  4. TF-IDF Vectorization: Converts text to numerical features
  5. Classification: Predicts spam or legitimate
  6. Confidence Scoring: Shows prediction certainty

📊 Model Details

Algorithm: Naive Bayes Classifier

ML Framework: scikit-learn 1.3+

NLP Processing: NLTK

Feature Extraction: TF-IDF Vectorizer

Training Data: SMS Spam Collection Dataset

Total Features: 5000+ dimensions
